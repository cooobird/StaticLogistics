# Static Logistics 开发任务清单

## 1. 传输基础 (Transfer Foundation)
* **资源类型支持**
  * [x] **物品 (Items)**：通过 IItemHandler 实现。
  * [x] **流体 (Fluids)**：通过 IFluidHandler 实现。
  * [x] **能量 (Energy/FE)**：支持 FE 能量调度。
  * [ ] 化学品 (Chemicals) - 计划对接 Mekanism
  * [ ] 魔源 (Essentia) - 计划对接 新生魔艺
  * [ ] 魔力 (Mana) - 计划对接 Botania
* **传输逻辑**
  * [x] **点对点硬链接**：基于物理坐标的 StaticLink 映射，不依赖逻辑频道。
  * [x] **多维拓扑结构**：支持 1:N、N:1 和网格化链路。
  * [x] **分发策略**：集成轮询、顺序、随机及距离优先算法。

## 2. 逻辑与过滤 (Logic & Filters)
* **配置功能**
  * [x] **面独立控制**：6 个面可独立设置输入/输出模式。
  * [x] **基础过滤**：物品黑白名单过滤。
  * [ ] **自定义染色**：计划在 GUI 面配置界面加入，用于改变逻辑连线的渲染颜色（仅视觉用途）。
* **高级控制**
  * [ ] **红石信号控制**：通过红石信号实时开启/关闭链路传输。
  * [ ] **强力 NBT 匹配**：过滤系统中加入 NBT 深度校验。

## 3. 交互与视觉 (Interaction & Visuals)
* **工具增强 (Link Configurator)**
  * [x] **三模链接器**：连接、删除、配置模式已完成。
  * [ ] **远程配置**：无需点击方块，直接在工具界面修改链路参数。
* **视觉渲染**
  * [x] **基础链路渲染**：已实现基于 TransferType 的硬链接连线渲染（LinkWorldRenderer）。
  * [x] **操作高亮**：已实现源面（绿色）与目标面（橙色）的叠加层显示。
  * [ ] **动态效果**：计划为连线增加流动动画或粒子效果以区分流向。

## 4. 优化与兼容 (Optimization & Compatibility)
* [x] **智能休眠**：源容器为空时自动进入冷却模式。
* [x] **性能缓存**：深度集成 BlockCapabilityCache 优化查询。
* [x] **权限集成**：原生兼容 FTB Teams。
* [ ] **并发调度**：针对超大规模物流网的计算优化。
